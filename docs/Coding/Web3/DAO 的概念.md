去中心化分布式的记账系统

**区块**：多条账目的打包  
**链**：将两个相邻的日子的账本联系起来  
**区块链**：大家共同维护一个账本，并且这个账本具有不可被随意篡改、公开透明、账目可靠和去中心化的特点  
区块链的网络路由方式，用户对用户，每个人既是信息的接收者，同时也是信息的传播者
